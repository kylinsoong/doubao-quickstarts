# 培训材料生成器

## 任务目标
- 本 Skill 用于：快速生成结构化的Word格式培训材料
- 能力包含：
  - 根据培训主题和听众特征设计材料结构
  - 生成符合专业标准的培训内容
  - 输出格式化的Word文档
- 触发条件：用户需要准备培训材料，提供了培训主题和目标听众信息

## 前置准备
- 依赖说明：使用python-docx库生成Word文档
  ```
  python-docx>=0.8.11
  ```

## 操作步骤
1. **分析培训需求**
   - 确认培训主题的核心要点和目标
   - 分析听众特征（知识水平、背景、学习需求）
   - 参考培训材料结构标准

2. **设计材料结构**
   - 根据主题复杂度确定章节划分
   - 设计适合听众的内容深度
   - 规划每个章节的核心内容点

3. **生成结构化内容**
   - 使用自然语言生成完整的培训内容
   - 将内容组织为结构化JSON格式（见下文格式规范）
   - 确保内容逻辑连贯、层次清晰

4. **生成Word文档**
   - 调用 `scripts/create_training_docx.py` 处理
   - 传入参数：
     - `topic`: 培训主题
     - `audience`: 听众描述
     - `content_json`: 结构化内容（JSON字符串）
     - `output_path`: 输出文件路径（如 `./培训材料.docx`）
   - 脚本将生成格式化的Word文档

## 结构化内容格式

### JSON格式规范
生成的内容需符合以下JSON结构：

```json
{
  "title": "培训主标题",
  "subtitle": "培训副标题或描述",
  "author": "讲师/制作人",
  "chapters": [
    {
      "chapter_title": "第一章标题",
      "sections": [
        {
          "section_title": "小节标题",
          "content": "详细的段落内容，可以包含多个段落",
          "points": ["关键要点1", "关键要点2", "关键要点3"]
        }
      ]
    }
  ]
}
```

### 字段说明
- `title`: 培训材料的总标题
- `subtitle`: 可选，副标题或培训目标描述
- `author`: 可选，制作人信息
- `chapters`: 章节数组
  - `chapter_title`: 章节标题
  - `sections`: 小节数组
    - `section_title`: 小节标题
    - `content`: 该小节的详细内容（支持多个自然段）
    - `points`: 可选，关键要点列表

## 注意事项
- 确保JSON格式正确，避免语法错误
- 根据听众水平调整内容深度和专业术语使用
- 每个章节的内容应保持适中的长度，避免过长
- 使用简洁清晰的语言表达，提高可读性

## 使用示例

### 示例1：企业新员工入职培训
- **培训主题**: 新员工企业文化与制度培训
- **听众**: 企业新入职员工
- **执行方式**:
  1. 智能体分析新员工特点和培训目标
  2. 设计包含企业文化、组织架构、规章制度等章节
  3. 生成适合新员工理解的详细内容
  4. 调用脚本生成Word文档

### 示例2：技术技能培训
- **培训主题**: Python数据分析入门
- **听众**: 有一定编程基础的数据分析初学者
- **执行方式**:
  1. 智能体设计从基础到进阶的学习路径
  2. 生成包含理论讲解和实例的章节
  3. 突出实用代码示例和实践要点
  4. 调用脚本生成Word文档

### 示例3：管理技能提升培训
- **培训主题**: 团队协作与沟通技巧
- **听众**: 中层管理人员
- **执行方式**:
  1. 智能体设计理论与实践结合的结构
  2. 生成包含案例分析、方法论的内容
  3. 强调实际应用场景和技巧要点
  4. 调用脚本生成Word文档
