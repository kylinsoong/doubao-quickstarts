# 乳腺癌：早发现、早治疗的科学指南

## 一、疾病概述
### 1.1 什么是乳腺癌？
乳腺癌（Breast Cancer）是起源于乳腺上皮细胞的恶性肿瘤，全球女性最高发的癌症类型。根据WHO 2022年数据，每年新发病例超230万例，中国国家癌症中心统计显示，我国女性乳腺癌发病率以年均3-4%速度增长。

![乳腺癌发病部位示意图](https://via.placeholder.com/600x400?text=乳腺解剖结构)

### 1.2 高危人群
| 风险因素        | 具体说明                      |
|-----------------|-----------------------------|
| 遗传因素        | BRCA1/2基因突变携带者风险提升5倍 |
| 激素暴露        | 初潮<12岁、绝经>55岁         |
| 生育史          | 未育或初产>35岁              |
| 生活方式        | 长期饮酒、肥胖（BMI>30）      |

## 二、发病机制
### 2.1 分子分型
根据2022 CSCO指南分为四型：
1. **Luminal A型**（ER+/PR+/HER2-）：占比40-50%
2. **Luminal B型**（ER+/PR±/HER2+）
3. **HER2阳性型**
4. **三阴性型**（TNBC）：易转移，预后较差

### 2.2 关键信号通路
* PI3K-AKT-mTOR通路异常激活
* ERα介导的雌激素信号传导
* HER2受体过表达

## 三、临床表现
### 3.1 典型症状
```markdown
1. **无痛性肿块**（80%首发症状）
2. 皮肤改变：橘皮征、酒窝征
3. 乳头溢液（血性需特别注意）
4. 腋窝淋巴结肿大
```

### 3.2 影像学特征
| 检查方法 | 优势 | 局限性 |
|----------|------|--------|
| 乳腺超声 | 无辐射，区分囊实性 | 依赖操作者经验 |
| 钼靶检查 | 钙化灶敏感 | 致密乳腺检出率低 |
| MRI      | 多参数成像 | 费用高、假阳性多 |

## 四、诊断流程
### 4.1 三阶梯诊断法
1. 临床检查 → 2. 影像评估 → 3. 病理确诊

### 4.2 新型检测技术
* 液体活检（ctDNA监测）
* 人工智能辅助诊断（AI-CAD系统）
* 18F-FES PET显像（雌激素受体成像）

## 五、治疗方案
### 5.1 手术治疗演进
```markdown
- 传统根治术 → 保乳手术（需满足：肿瘤<3cm，距乳头>2cm）
- 前哨淋巴结活检替代腋窝清扫
- 乳房重建技术（假体/自体组织）
```

### 5.2 系统治疗进展
1. **靶向药物**：
   - HER2阳性：曲妥珠单抗（赫赛汀）
   - CDK4/6抑制剂（帕博西林）
   
2. **免疫治疗**：
   - PD-1抑制剂（Keytruda）用于TNBC

3. **ADC药物**：
   - DS-8201（Enhertu）突破HER2低表达治疗

## 六、预防策略
### 6.1 筛查指南
| 年龄段 | 筛查建议 |
|--------|----------|
| 20-39岁 | 每月自检 + 临床体检 |
| 40-69岁 | 年度超声 + 每2年钼靶 |
| >70岁   | 个体化评估 |

### 6.2 预防性措施
1. 基因检测（BRCA突变者可行预防性切除）
2. 药物预防（他莫昔芬风险降低38%）
3. 生活方式干预（运动5次/周，每次30分钟）

## 七、康复管理
### 7.1 淋巴水肿防护
* 术后患肢血压监测
* 渐进式负重训练
* 气压循环治疗仪

### 7.2 心理康复
建立多学科支持体系：
- 心理咨询
- 病友互助小组
- 性健康指导

## 八、最新研究（2023）
1. **疫苗研发**：
   - Moderna mRNA疫苗进入II期临床
   
2. **AI应用**：
   - 深度学习预测新辅助化疗反应（准确率92%）

3. **代谢疗法**：
   - 二甲双胍联合方案降低复发风险

> **特别提示**：乳房自查应选择月经后7-10天，采用「指腹按压法」按顺时针方向系统触诊。
