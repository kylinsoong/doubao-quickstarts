<a name="早期预警信号"></a>
## 3. 早期预警信号

### 3.1 自检四步法
1. **视诊**：对镜观察乳房轮廓变化
2. **触诊**：三指腹螺旋式触压
3. **挤压**：检查乳头溢液
4. **对比**：双侧乳房对比检查

### 3.2 临床表现谱
| 症状类型       | 典型表现                     | 危险等级 |
|----------------|------------------------------|----------|
| 肿块特征       | 无痛、质硬、边界不清         | ★★★★     |
| 皮肤改变       | 橘皮样变、酒窝征             | ★★★★     |
| 乳头异常       | 内陷、溢血、糜烂             | ★★★★     |
| 淋巴结肿大     | 腋窝/锁骨上结节              | ★★★      |
