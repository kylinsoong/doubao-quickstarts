# 乳腺癌：从预防到康复的全面科普指南

## 概述
乳腺癌是女性最常见的恶性肿瘤之一，占全球女性癌症发病率的24.2%（WHO 2023数据）。中国国家癌症中心统计显示，每年新发病例约42万，且呈现年轻化趋势。男性乳腺癌占比不足1%，但近年检出率有所上升。

## 一、病因与高危因素
### 1.1 不可改变因素
- **遗传基因**：BRCA1/2基因突变携带者终生风险达60-80%
- **性别与年龄**：40-50岁、60-70岁为双峰期
- **月经周期**：初潮<12岁或绝经>55岁风险↑30%

### 1.2 可干预因素
- **激素暴露**：长期口服避孕药（>5年）风险↑20%
- **生活方式**：每天饮酒50g风险↑40%（NEJM研究）
- **代谢异常**：BMI>30绝经后风险↑50%
- **生育因素**：未生育或首胎>35岁风险↑30%

### 3.2 病理诊断流程
1. 影像学发现BI-RADS 4类以上病灶
2. 空心针穿刺活检（CNB）
3. 免疫组化检测ER/PR/HER2/Ki-67
4. 21基因检测（指导治疗方案）

## 四、精准治疗策略
### 4.1 外科手术进展
- **保乳手术**：适应症扩大至肿瘤≤3cm且切缘阴性
- **前哨淋巴结活检**：替代传统腋窝清扫，淋巴水肿率↓70%

### 4.2 系统治疗
- **靶向药物**：曲妥珠单抗（HER2+）使5年生存率↑至90%
- **CDK4/6抑制剂**：HR+/HER2-晚期患者PFS延长10个月
- **ADC药物**：DS-8201对HER2低表达有效

### 4.3 放疗新技术
- 术中放疗（IORT）：单次20Gy替代传统6周方案
- 质子治疗：心肺受量减少50%
