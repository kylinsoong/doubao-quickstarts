# 乳腺癌：早筛早治与科学应对指南

## 一、概述与流行病学
### 1.1 疾病定义
乳腺癌（Breast Cancer）是起源于乳腺导管/小叶上皮的恶性肿瘤，中国国家癌症中心2023年数据显示：
- 年新发患者42.9万例（占女性恶性肿瘤24.2%）
- 年均增速达3%-4%，城市发病率显著高于农村
- 发病年龄较欧美提前10-15年，45-55岁为高发区间
