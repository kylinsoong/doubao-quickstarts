# 乳腺癌：科学认知与防治指南

## 一、乳腺癌概述
### 1.1 定义与流行病学
乳腺癌（Breast Cancer）是乳腺上皮细胞在多种致癌因子作用下发生增殖失控的恶性肿瘤。据WHO最新数据：
- 全球每年新发病例约 **230万例**（占女性恶性肿瘤24.5%）
- 我国年新发患者 **42万**，增速达3-4%/年
- 20-40岁年轻患者占比上升至 **15%**

### 1.2 疾病发展特征
| 分期 | 肿瘤大小 | 淋巴转移 | 远端转移 |
|------|---------|----------|----------|
| 0期 | 原位癌 | 无       | 无       |
| I期 | ≤2cm   | 无       | 无       |
| II期| 2-5cm   | 腋下1-3个| 无       |
| III期| >5cm   | 腋下≥4个 | 无       |
| IV期| 不限    | 不限     | 存在     |

## 二、高危因素与预防策略
### 2.1 不可控风险因素
- 遗传基因：BRCA1/2突变携带者终身风险达 **70%**
- 性别与年龄：女性风险是男性100倍，45-55岁为发病高峰
- 初潮早（<12岁）/绝经晚（>55岁）

### 2.2 可控生活方式干预
- 肥胖（BMI>30风险增加50%）
- 长期酗酒（每日1杯风险提升7-10%）
- 缺乏运动（每周≥150分钟中强度运动降低20%风险）
- 激素替代治疗（使用>5年风险增加30%）

### 2.3 三级预防体系
1. 一级预防：基因检测+生活方式管理
2. 二级预防：40岁起每1-2年乳腺X线筛查
3. 三级预防：规范治疗+康复管理

## 三、临床表现与诊断技术
### 3.1 典型症状
> 注意：出现以下症状应立即就诊：
> - 无痛性乳房肿块（83%患者首发症状）
> - 乳头溢液（血性液体需警惕）
> - 皮肤橘皮样改变
> - 乳头凹陷或偏移

### 3.2 诊断金标准
1. **乳腺X线摄影（钼靶）**：检出微钙化灶（灵敏度85%）
2. **超声检查**：鉴别囊实性肿块（尤其适合致密型乳腺）
3. **MRI**：用于高危人群筛查（灵敏度95%）
4. **穿刺活检**：病理诊断最终依据

### 3.3 分子分型指导治疗
| 分型           | 占比   | 特征               | 治疗方案           |
|----------------|--------|--------------------|--------------------|
| Luminal A型    | 50-60% | ER/PR阳性，HER2阴性 | 内分泌治疗为主     |
| HER2过表达型   | 15-20% | HER2阳性           | 靶向药物（曲妥珠单抗）|
| 三阴性         | 10-15% | 三者均阴性         | 化疗为主           |

## 四、精准治疗新进展
### 4.1 外科治疗革新
- 保乳手术（适应症扩大至肿瘤≤3cm）
- 前哨淋巴结活检（减少淋巴水肿发生率）
- 乳房重建技术（假体/自体组织移植）

### 4.2 系统治疗方案
- **新辅助化疗**：使不可手术转为可手术
- **CDK4/6抑制剂**：Palbociclib联合内分泌治疗
- **免疫治疗**：PD-1抑制剂用于三阴性乳腺癌
- **ADC药物**：DS-8201（HER2低表达患者获益）

### 4.3 放疗技术升级
- 术中放疗（单次大剂量照射）
- 质子治疗（精准保护心肺器官）
- 短程放疗（1周替代传统6周方案）

## 五、康复管理与生存质量
### 5.1 术后康复关键期
- 淋巴水肿预防：术后6个月内专业康复训练
- 心理干预：30%患者出现焦虑抑郁需专业疏导
- 性健康管理：激素治疗导致阴道干涩等问题的解决方案

### 5.2 长期随访计划
- 治疗后2年内：每3个月复查
- 3-5年：每6个月复查
- 5年后：每年全面检查

### 5.3 营养与运动建议
- 地中海饮食模式（降低复发风险41%）
- 阻力训练+有氧运动（每周≥150分钟）
- 维生素D补充（维持血清浓度≥30ng/mL）
