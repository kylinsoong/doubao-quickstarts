# 乳腺癌：从预防到治疗的全面科普指南

## 概述
乳腺癌（Breast Cancer）是乳腺上皮细胞在致癌因素作用下发生增殖失控的恶性肿瘤，全球发病率位居女性恶性肿瘤首位。据WHO 2022年数据显示：
- 全球每年新发病例约230万例
- 占所有新发癌症病例的11.7%
- 死亡率位居女性癌症死亡原因第5位

（示意图：近20年全球乳腺癌年龄标化发病率变化）

## 病因与风险因素

### 不可改变因素
| 风险因素 | 影响机制 | 风险倍数 |
|---------|---------|---------|
| 女性性别 | 雌激素暴露时间长 | 男性发病率仅为女性1% |
| 年龄增长 | 基因突变累积 | ＞50岁风险骤增 |
| BRCA突变 | DNA修复缺陷 | 携带者风险达60-80% |

### 可干预因素
- **代谢相关**  
  - 肥胖（BMI＞30风险增加20%）  
  - Ⅱ型糖尿病（风险升高23%）

- **生殖因素**  
  - 初潮＜12岁（风险+20%）  
  - 未生育或首胎＞35岁（风险+40%）

- **生活方式**  
  - 每周饮酒＞3次（风险增加15%）  
  - 久坐＞6小时/天（风险升高10%）

## 临床表现与诊断

### 早期警示信号
```diff
+ 无痛性乳房肿块（82%患者首发症状）
+ 乳头血性溢液（尤其单侧单孔）
+ 皮肤橘皮样改变
- 注意：约6%患者以腋窝淋巴结肿大为首发表现
```

### 诊断金三角
1. **影像学检查**
   - 超声（首选＜40岁，敏感性92%）
   - 钼靶（＞40岁筛查，可发现微钙化）
   - MRI（高危人群，敏感性98%）

2. **病理学确诊**
   - 空心针穿刺活检（准确率＞95%）
   - 免疫组化检测ER/PR/HER2

3. **分子分型**

    Basal-like(基底样型) --> 三阴性

## 精准治疗体系

### 分期治疗原则
| 分期 | 治疗方案 | 5年生存率 |
|------|---------|-----------|
| 0期（DCIS） | 保乳手术+放疗 | 99% |
| I期 | 手术±化疗 | 98-100% |
| II期 | 新辅助治疗+手术 | 85-90% |
| III期 | 综合治疗 | 40-70% |
| IV期 | 姑息治疗 | 22% |

### 靶向治疗突破
- **HER2阳性**：曲妥珠单抗（使复发风险降低52%）
- **CDK4/6抑制剂**：Palbociclib（转移性HR+患者PFS延长10个月）
- **PARP抑制剂**：奥拉帕利（BRCA突变患者OS达19.3个月）

## 预防策略

### 三级预防体系
1. **一级预防**
   - 哺乳＞12个月（风险降低28%）
   - 地中海饮食（风险降低15%）
   - 每周150分钟中等强度运动

2. **二级预防**
   - 40岁起每2年钼靶筛查（降低死亡率20%）
   - 高危人群年度MRI检查

3. **三级预防**
   - 规范化疗降低50%复发风险
   - 内分泌治疗持续5-10年

## 研究前沿
1. **液体活检**：ctDNA检测实现0.1%微小残留病灶识别
2. **疫苗研发**：GP2肽疫苗Ⅱ期试验预防复发有效率100%
3. **人工智能**：Google AI钼靶阅片准确率超放射科医师11.5%
