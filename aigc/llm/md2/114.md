# 乳腺癌医学科普：从预防到康复的全周期管理

## 概述
乳腺癌（Breast Cancer）是全球女性最常见的恶性肿瘤。据WHO统计，2020年全球新发病例达226万例，占所有癌症病例的11.7%。中国国家癌症中心数据显示，我国每年新发乳腺癌约42万例，发病率以每年3%-4%的速度递增。但随着医学进步，早期乳腺癌的5年生存率已超过90%。

---

## 一、病因与风险因素

### 1.1 核心发病机制
乳腺癌本质是乳腺上皮细胞发生恶性增殖，其发展涉及多阶段演变：
```
正常细胞 → 非典型增生 → 原位癌 → 浸润癌
```

### 1.2 主要风险因素
| 风险等级 | 具体因素 |
|---------|---------|
| 高度相关 | BRCA1/2基因突变、胸部放射治疗史 |
| 中度相关 | 初潮<12岁、绝经>55岁、未生育或晚育（>35岁） |
| 可控因素 | 肥胖（BMI>30）、长期饮酒（>15g/天）、缺乏运动 |

## 二、临床表现与诊断

### 2.1 典型症状（BIRADS分类指南）
- **触诊异常**：无痛性肿块（91%病例首发症状）
- **皮肤改变**：橘皮样变、酒窝征
- **乳头异常**：血性溢液（5-10%为恶性）、乳头内陷
- **影像学特征**：微小钙化灶（恶性可能达80%）

## 四、预防与筛查

### 4.1 三级预防体系
1. **一级预防**：基因检测（BRCA突变者预防性切除可使风险降90%）
2. **二级预防**：40岁起年度钼靶+超声（高危人群提前至25岁）
3. **三级预防**：规范治疗+全程管理

### 4.2 自查指南（5步法）
1. 观察：对镜查外形
2. 触摸：指腹螺旋按压
3. 挤压：查乳头溢液
4. 对比：双侧对照
5. 记录：绘制异常区域

## 五、康复管理

### 5.1 淋巴水肿防治
- **发生率**：腋窝清扫术后约20-30%
- **黄金期**：术后6周内开始理疗
- **压力治疗**：梯度压力袖套（23-32mmHg）

### 5.2 心理康复数据

|---------|-----------|-----------|

| 团体治疗 | 55% | 61% |
| 药物干预 | 40% | 45% |

### 5.3 营养管理要点

- **关键指标**：保持BMI 18.5-24.9
