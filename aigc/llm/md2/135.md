# 乳腺癌：从认知到防治的全方位科普

## 概述
乳腺癌是全球女性最常见的恶性肿瘤之一，据WHO统计，2020年新增病例达226万例，约占女性癌症总发病率的24.5%。中国国家癌症中心数据显示，我国每年新发乳腺癌病例约42万，发病高峰年龄在45-55岁之间。随着早筛技术提升，早期乳腺癌五年生存率已超过90%。

## 一、病因与危险因素

### 1.1 不可改变因素
- 🔴 遗传基因突变：BRCA1/BRCA2基因突变携带者终生风险达70%
- 🔴 性别与年龄：女性发病率是男性100倍，50岁以上风险显著增加
- 🔴 月经史：初潮<12岁或绝经>55岁风险提升2-3倍

### 1.2 可干预因素
- 🟡 激素暴露：长期HRT治疗（>5年）风险增加26%
- 🟡 生活方式：肥胖（BMI>30）风险升高20%
- 🟡 生育特征：未生育或首胎>35岁风险增加1.5倍

### 危险等级对照表

| 风险等级 | 10年发病概率 | 干预建议 |
|---------|-------------|----------|
| 平均风险 | <3%         | 常规筛查 |
| 中等风险 | 3-8%        | 加强监测 |
| 高风险  | >8%         | 预防用药 |

## 二、临床表现与诊断

### 2.1 典型症状
1. 无痛性乳房肿块（82%患者首发症状）
2. 乳头异常：溢液（尤其是血性）、内陷
3. 皮肤改变：橘皮征、酒窝征
4. 腋窝淋巴结肿大（晚期表现）

### 2.2 诊断金标准
1. **三合诊检查**：医生触诊+影像学检查
2. **影像学检查**：
   - 乳腺X线摄影（检出率85%）
   - 超声检查（敏感度90%）
   - MRI（高危人群首选）
3. **病理活检**：
   - 空芯针穿刺（准确率95%）
   - 术中冰冻切片

### 分子分型标准

| 分型       | 占比  | 治疗方案           |
|------------|-------|--------------------|
| Luminal A  | 50-60%| 内分泌治疗为主     |
| HER2阳性型 | 15-20%| 靶向+化疗          |
| 三阴性型   | 10-15%| 化疗+免疫治疗      |

## 三、综合治疗策略

### 3.1 手术方案选择
```markdown
- 保乳手术：肿瘤<3cm且切缘阴性，5年生存率与全切相当
- 改良根治术：局部晚期患者首选
- 乳房重建术：即刻/延期重建技术成熟
```

### 3.2 辅助治疗
1. 放疗：保乳术后必须，全切术后高危患者推荐
2. 化疗：AC-T方案（阿霉素+环磷酰胺序贯紫杉醇）
3. 靶向治疗：曲妥珠单抗（HER2阳性标准治疗）
4. 内分泌治疗：他莫昔芬/AI类药物（ER阳性患者）

### 治疗副作用管理
```markdown
- ✅ 化疗脱发：冰帽预防有效率60%
- ✅ 神经毒性：度洛西汀可缓解70%症状
- ✅ 骨质疏松：双膦酸盐+维生素D联合方案
```

## 四、预防与筛查指南

### 4.1 筛查建议
```markdown
- 40-44岁：高危人群年度筛查
- 45-54岁：全民年度钼靶检查
- 55+岁：两年一次筛查
```

### 4.2 预防措施
1. 生活方式干预：
   - 每周150分钟中等强度运动
   - 地中海饮食（降低风险19%）
2. 药物预防：
   - 他莫昔芬（高风险人群降低49%发病率）
3. 预防性手术：
   - BRCA突变者双侧乳腺切除术
