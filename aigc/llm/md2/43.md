# 乳腺癌医学科普指南

## 概述
### 疾病定义
乳腺癌是乳腺上皮细胞在多种致癌因素作用下发生的恶性肿瘤，全球女性发病率最高的癌症类型。中国年均新发病例约42万例（2023年数据），占女性恶性肿瘤24.2%。

### 流行病学特征
- 发病年龄：45-55岁为发病高峰
- 性别分布：女性占99%，男性病例约占1%
- 地域特征：城市发病率高于农村（2.5:1）
- 五年生存率：早期可达90%以上，晚期降至约30%

## 病因与风险因素
### 不可控因素
1. 遗传因素：BRCA1/BRCA2基因突变携带者风险提升5-8倍
2. 性别与年龄：女性风险是男性100倍，50岁后风险陡增
3. 月经周期特征：
   - 初潮年龄<12岁
   - 绝经年龄>55岁
4. 乳腺疾病史：非典型增生患者风险增加4-5倍

### 可控风险因素
- 生育特征：未生育或首胎>35岁
- 哺乳情况：未哺乳或哺乳期<6个月
- 激素暴露：长期口服避孕药或激素替代治疗
- 生活方式：
  - BMI>30的肥胖人群
  - 每周酒精摄入>14单位
  - 长期缺乏运动（每周<150分钟中等强度运动）

## 临床表现
### 典型症状
1. 乳房肿块（97%患者首发症状）
   - 质地：石样坚硬
   - 边界：不规则呈蟹足状
   - 活动度：早期可推动，晚期固定

2. 皮肤改变：
   - 橘皮样变
   - 酒窝征
   - 卫星结节

3. 乳头异常：
   - 血性溢液（12%病例）
   - 乳头回缩或偏位

### 转移症状
| 转移部位 | 临床表现                 |
|----------|--------------------------|
| 骨转移   | 病理性骨折、骨痛         |
| 肝转移   | 黄疸、腹水、肝区疼痛     |
| 肺转移   | 咳血、呼吸困难           |
| 脑转移   | 头痛、癫痫、神经功能障碍 |

## 诊断方法
### 影像学检查
1. 乳腺X线摄影（钼靶）
   - 敏感度：85%（50岁以上）
   - 特征表现：簇状钙化灶

2. 超声检查
   - 优势：鉴别囊实性肿块
   - BI-RADS分级：
     - 4类以上需活检

3. 增强MRI
   - 适用人群：
     - 致密型乳腺
     - 假体植入者

### 病理学诊断
```markdown
1. 穿刺活检（核心针穿刺）
   - 准确率：92-97%
2. 免疫组化检测：
   - ER/PR（激素受体）
   - HER-2（人表皮生长因子受体2）
   - Ki-67（增殖指数）
```

## 治疗体系
### 手术治疗
```markdown
1. 保乳手术
   - 适应证：
     - 肿瘤≤3cm
     - 单发病灶
     - 术后可接受放疗
2. 全乳切除
   - 适应证：
     - 多中心病灶
     - 无法保留乳头乳晕复合体
```

### 全身治疗
| 治疗类型       | 代表药物               | 适用人群特征               |
|----------------|------------------------|---------------------------|
| 化疗           | 多西他赛+环磷酰胺      | 三阴性乳腺癌              |
| 内分泌治疗     | 他莫昔芬/芳香酶抑制剂  | ER/PR阳性                 |
| 靶向治疗       | 曲妥珠单抗             | HER-2阳性                 |
| 免疫治疗       | PD-1抑制剂             | PD-L1高表达               |

### 放射治疗
- 保乳术后常规全乳照射（50Gy/25次）
- 新技术：
  - 术中放疗（单次20Gy）
  - 质子治疗（减少心肺照射）

## 预防与筛查
### 三级预防体系
1. 一级预防
   - 控制体重（BMI<24）
   - 每周中等强度运动≥150分钟
   - 母乳喂养≥6个月

2. 二级预防
   - 筛查方案：
     - 40-44岁：高危人群每年钼靶
     - 45-69岁：每1-2年钼靶+超声
   - 自查方法（月经后7-10天）：
     - 观察法：对镜查外形
     - 触诊法：三指螺旋式按压

3. 三级预防
   - 规范随访：
     - 术后2年：每3月复查
     - 3-5年：每6月复查
     - 5年后：每年复查

## 最新研究进展（2023-2024）
1. 新型ADC药物
   - DS-8201（Enhertu）：HER-2低表达人群OS提升33%

2. 液体活检技术
   - ctDNA监测微小残留病灶（MRD）
   - 准确率：88%（Ⅲ期临床试验数据）

3. AI辅助诊断
   - 深度学习模型：
     - 钼靶读片敏感度提升至96%
     - 假阳性率降低40%

## 总结
乳腺癌是可防可治的恶性肿瘤，通过规范筛查可实现早诊早治。建议高危人群建立个性化防控方案，确诊患者应进行分子分型指导下的精准治疗。随着ADC药物、免疫治疗等新疗法不断突破，晚期患者生存期已显著延长。
